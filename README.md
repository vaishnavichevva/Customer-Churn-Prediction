# Customer Churn Prediction

A machine learning project that predicts whether a customer is likely to leave a service using historical customer data. It applies classification algorithms to analyze customer behavior, identify key churn factors, and support data-driven customer retention strategies.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Classification model training
- Model evaluation using accuracy, precision, recall, and F1-score
- Customer churn prediction

## Dataset
The dataset contains customer information such as demographics, account details, service usage, and churn status. It is used to train and evaluate classification models.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/customer-churn-prediction.git
   ```
2. Navigate to the project folder:
   ```bash
   cd customer-churn-prediction
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the Jupyter Notebook or Python script to preprocess the data, train the model, evaluate its performance, and generate churn predictions.

## Results
The trained classification model predicts whether a customer is likely to churn, helping businesses identify at-risk customers and improve retention strategies.
